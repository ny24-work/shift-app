勤務予定表 自動作成 完成版 Ver.4.1

機能
- 店長・西岡・スタッフの役割分け
- 開発者名は「西岡・開発者」
- 今月・翌月・翌々月の3ヶ月表示
- 希望休・食数・勤務表を3ヶ月分登録
- A〜F勤務設定
- 食数から必要人数を自動計算
- 休み希望を反映して自動シフト作成
- 最大連勤・月上限日数
- タイミー不足表示
- 勤務日数・勤務時間集計
- 勤務表セルを直接編集可能
- 印刷/PDF保存
- CSV出力
- 店長/西岡のパスワード変更
- バックアップJSON出力/読込
- PWA対応

初期パスワード
店長：0000
西岡：9999
スタッフ：空欄

GitHub更新方法
1. ZIPを解凍
2. 中の7ファイルを全部GitHubへ上書きアップロード
3. Commit changes
4. 公開URLの最後に ?v=4 を付けて開く

- スタッフは希望休と勤務表だけ表示
- パスワードのヒント表示を削除
